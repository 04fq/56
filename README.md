# Installation 🎩
## Linux 🐧
* cd Desktop
* git clone https://github.com.04fq/56.git
* cd 56
* pip install -r requirement.txt
* python 56.py  
## Android (Termux) 👽
* git clone https://github.com.04fq/56.git
